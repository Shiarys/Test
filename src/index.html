Как коммитить:

git init

Инициация в репозиторий

git config

Указание своего имени, глобально или локально

git config --local/global user.name "Orlova"
git config --local/global user.email shiarys@yandex.ru
глобальные не распространяются на локальные. Глобальные - на все проекты на ПК.

git status

статус

git add

добавление файлов в индекс(индексация в git).

git add -A "A" означает, что добавляются все файлы
git add *.расширениеФайлов - все файлы с нужным расширением
git add index.html - название конкретного файла

git commit

создание точки сохранения

git commit -a -m"Название коммита" // -a=all файлы, -m - сообщение - название коммита

git log

git remote add origin ссылкаНаРепозиторий

добавление ссылки на существующий репозиторий

git push -u origin main/master
